# moviefront

## 개요
영화 커뮤니티 사이트 MovieSite의 프론트엔드 애플리케이션(2024.04 ~ )

## 개발환경
* 개발 언어 : JavaScript
* 프레임워크 : Vuejs(version 3)

## 배포 환경(예정)
+ Amazon EC2 인스턴스
    - 운영체제 : Amazon Linux 2
    - 인스턴스 유형 : t2.micro

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
